Pd Shortcuts

---------

CMD/CTRL + "" = does something!

CMD/CTRL + E  = unlocks/locks patch for editing
CMD/CTRL + 1  = new object [osc~]
CMD/CTRL + 2  = new message ["helloworld"(
CMD/CTRL + 3  = number atom box
CMD/CTRL + 4  = symbol box
CMD/CTRL + 5  = comment

CMD/CTRL + / = switches on audio
CMD/CTRL + . = switches off audio
